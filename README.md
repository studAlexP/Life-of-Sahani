## Setup
To run this project, install it locally using npm:
```
# Clone this repository
$ https://github.com/studAlexP/Life-of-Sahani.git

# Go into the repository
$ cd Life-of-Sahani

# Install dependencies
$ npm install

# Run the app
$ npm start
```
